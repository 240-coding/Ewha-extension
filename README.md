# 이화 바로가기

`이화 바로가기` 는 이화여대 학생들의 편의를 위해 이화여자대학교와 관련된 다양한 사이트 링크를 모아둔 웨일 확장앱입니다.
학사일정, 수강신청 등 기본적인 사이트뿐만 아니라 내 단과대, 학과를 선택하는 기능도 제공하여 나에게 꼭 필요한 바로가기들을 더욱 편리하게 이용할 수 있습니다.
